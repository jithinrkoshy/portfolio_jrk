This website portrays a hypothetical organisation working for the conservation of trees. As a part of their process, this organisation maintains some data about forests, mainly trees, in the website. Anyone visiting the site can navigate through the list of trees and choose any tree. This will produce some data related to that tree along with some pictures. This website is developed using HTML, CSS, Javascript and Python. Django, a web development framework, has been used for the development.
